Classpublic class FaceRecognition
InheritanceFaceRecognition flash.events.EventDispatcher

Główna klasa w tym pakiecie. Używana jest do rozpoczęcia ładowania zdjeć do treningu systemu oraz samego treningu z wykorzystaniem tych zdjęć. Za jej pośrednictwem inicjowany jest również proces rozpoznawania nie sklasyfikowanej twarzy z wykorzystaniem metody wykorzystującej koncepcje Eigenfaces.

probe()method 
public function probe(f:Face, classthreshold:int = 5, numVecs:int = 10):Face

Idefikuje twarz i zwraca obiekt klasy Face przekazany jako parametr ale z zaktualizowanymi wartościami classification oraz description. Identyfikator przypuszczalnego właściciela twarzy jest w classification.

Parameters
f:Face — Obiekt ze zdjeciem twarzy, która ma zostać poddana identyfikacji.
 
classthreshold:int (default = 5) — Ilość obiektów klasy FeatureVector używanych do porównania (zestaw treningowy musi mieć conajmniej tyle zdjęć co wartość podana w tym parametrze).
 
numVecs:int (default = 10) — Ilość wektorów. Ta sama wartość musi zostać podana przy wywołaniu funkcji this.train().

Returns
Face — Obiekt klasy Face przekazany jako parametr ale z zaktualizowaną wartościa classification oraz description. Identyfikator przypuszczalnego właściciela twarzy jest w probe().classification.
train()method 
public function train(numVecs:int = 10):void

Trenuje system używając do tego celu załadowanych zdjęć treningowych (do ładowania zdjeć można użyć funkcji this.loadTrainingFaces) oraz tekstów identyfikujących właścicieli tych twarzy.

Parameters
numVecs:int (default = 10) — Ilość tworzonych wektorów. Ta sama wartoś powinna zostać użyta dla tego parametru co dla parametru w funkcji this.probe().
