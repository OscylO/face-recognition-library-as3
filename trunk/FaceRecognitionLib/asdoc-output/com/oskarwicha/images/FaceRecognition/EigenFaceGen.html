getEigenFaces()method
public function getEigenFaces(pic:Picture, number:int):Array

Zwraca wartości eigenface dla zdjęcia twarzy przekazangeo za pomocą parametru pic. Zwracane wartości używane są w feature space czyli przestrzeni cech.

Parameters
pic:Picture — Zdjęcie twarzy
 
number:int — Ilość obliczanych wartości eigen faces.

Returns
Array — Wartości eigen face jako obiekty typu Number w tabeli o długości równej wartości parametru number lub this.getNumEigenVecs w zależności, która z tych wartości jest mniejsza
